# PyramidBox代码解析电子书

《PyramidBox代码解析》是首发于GiantPandaCV公众号的电子书教程，由zzk总结并撰写。Pyramid是百度研发的人脸检测深度学习算法，该算法在世界最权威的人脸检测数据集WIDER FACE拿下了榜首，刷新最好成绩。

代码链接：https://github.com/EricZgw/PyramidBox

论文链接：https://arxiv.org/abs/1803.07737
